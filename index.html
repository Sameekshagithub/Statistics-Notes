1Descriptive Statistics
Measures of Central Tendency
Mean (μ)

Sum of all values divided by count. Sensitive to outliers.

μ = (Σxᵢ) / n
Median

Middle value when sorted. Robust to outliers. Preferred for skewed data.

Mode

Most frequently occurring value. Can be multi-modal. Used in categorical data.

Weighted Mean

Each value has a weight. Critical in ensemble models.

μ_w = Σ(wᵢxᵢ) / Σwᵢ
Measures of Spread
Variance (σ²)
σ² = Σ(xᵢ - μ)² / n
Standard Deviation (σ)
σ = √[Σ(xᵢ - μ)² / n]
Interquartile Range (IQR)
IQR = Q3 - Q1
Coefficient of Variation
CV = (σ / μ) × 100%
Shape Descriptors

Skewness: Measures asymmetry. Positive skew → tail right. Negative skew → tail left.

Kurtosis: Measures tail heaviness. Leptokurtic (>3) = fat tails. Platykurtic (<3) = thin tails.

Skewness = E[(X - μ)³] / σ³    |    Kurtosis = E[(X - μ)⁴] / σ⁴
AI Relevance: Feature scaling (standardization, normalization), outlier detection, and data cleaning all rely on descriptive statistics.
2Probability Theory
Fundamentals
Sample Space ΩEvent A ⊆ Ω0 ≤ P(A) ≤ 1P(Ω) = 1
Addition Rule: P(A ∪ B) = P(A) + P(B) - P(A ∩ B)
Complement: P(Aᶜ) = 1 - P(A)
Conditional Probability & Independence
P(A | B) = P(A ∩ B) / P(B)    [B ≠ ∅]

A and B are independent if P(A ∩ B) = P(A)·P(B). Naive Bayes classifier assumes feature independence.

Bayes' Theorem
P(A | B) = P(B | A) · P(A) / P(B)
Posterior = (Likelihood × Prior) / Evidence

Fundamental to Bayesian learning, spam filters, medical diagnosis, and probabilistic graphical models.

Law of Total Probability
P(B) = Σᵢ P(B | Aᵢ) · P(Aᵢ)    where {Aᵢ} partition Ω
Expectation & Variance
E[X] = Σ xᵢ P(xᵢ)  (discrete)
E[X] = ∫ x f(x) dx  (continuous)
Var(X) = E[X²] - (E[X])²
Var(aX+b) = a² Var(X)

Used in Gaussian Mixture Models, LDA, Gaussian Processes. Covariance matrix Σ encodes feature correlations.

Central Limit Theorem (CLT)
X̄ ~ N(μ, σ²/n) as n → ∞

Sample means of any distribution converge to Normal as n grows. Justifies many ML approximations and mini-batch gradient descent.

4Statistical Inference
Point Estimation

A statistic θ̂ used to estimate unknown parameter θ.

Unbiasedness: E[θ̂]=θ Consistency: θ̂ →p θ as n→∞ Efficiency: min variance
Maximum Likelihood Estimation (MLE)
θ̂_MLE = argmax_θ L(θ; x) = argmax_θ Σᵢ log p(xᵢ | θ)

Foundation of most ML training. Minimizing cross-entropy loss ≡ maximizing log-likelihood for classification.

Maximum A Posteriori (MAP)
θ̂_MAP = argmax_θ [log L(θ;x) + log p(θ)]

MLE + prior regularization. L2 regularization corresponds to Gaussian prior; L1 to Laplace prior.

Confidence Intervals
CI: x̄ ± z_(α/2) · (σ/√n)    or    x̄ ± t_(α/2, n-1) · (s/√n)

A 95% CI means: 95% of such intervals (from repeated sampling) contain the true parameter.

Bias-Variance Tradeoff
MSE(θ̂) = Var(θ̂) + [Bias(θ̂)]²
High Bias (Underfitting)

Model too simple. Large training & test error. More features or model complexity needed.

High Variance (Overfitting)

Model too complex. Low training, high test error. Regularization, dropout, more data needed.

5Hypothesis Testing
Framework
H₀ (Null Hypothesis)

Default assumption. No effect, no difference. We try to reject this.

H₁ (Alternative Hypothesis)

What we want to prove. There is an effect or difference.

Errors & Significance
TypeDescriptionProbability
Type I (α)Reject true H₀ (False Positive)α = significance level
Type II (β)Fail to reject false H₀ (False Negative)β
Power (1-β)Correctly reject false H₀1 - β
p-value

Probability of observing results as extreme as the data, assuming H₀ is true. Reject H₀ if p < α (usually 0.05).

Important: p-value is NOT the probability that H₀ is true. A low p-value only tells us the data is inconsistent with H₀.

A/B Testing in ML

Used to compare model versions, recommendation algorithms, or UI changes. Requires careful sample size calculation:

n = 2 · (z_α/2 + z_β)² · σ² / δ²    (δ = desired effect size)
6Regression Analysis
Simple Linear Regression
y = β₀ + β₁x + ε    where ε ~ N(0, σ²)
OLS Estimators
β̂₁ = Σ(xᵢ-x̄)(yᵢ-ȳ) / Σ(xᵢ-x̄)²
β̂₀ = ȳ - β̂₁x̄
R² (Coefficient of Determination)
R² = 1 - SS_res/SS_tot = 1 - Σ(yᵢ-ŷᵢ)²/Σ(yᵢ-ȳ)²
Multiple Linear Regression
y = Xβ + ε    β̂ = (XᵀX)⁻¹Xᵀy

Assumptions: linearity, homoscedasticity, no multicollinearity, normality of residuals (GAUSS-MARKOV).

Regularization
Ridge Regression (L2)
min ‖y - Xβ‖² + λ‖β‖²

Shrinks coefficients. No feature selection. β̂ = (XᵀX + λI)⁻¹Xᵀy

Lasso Regression (L1)
min ‖y - Xβ‖² + λ‖β‖₁

Produces sparse solutions. Automatic feature selection. No closed form.

Elastic Net
min ‖y - Xβ‖² + λ₁‖β‖₁ + λ₂‖β‖²

Combines L1 and L2. Good when features are correlated and sparse solution is desired.

Logistic Regression (Classification)
P(y=1|x) = σ(xᵀβ) = 1/(1 + e^(-xᵀβ))
Loss: L = -Σ[yᵢ log(p̂ᵢ) + (1-yᵢ) log(1-p̂ᵢ)]  (Cross-Entropy)
7Bayesian Statistics
Core Principle
Posterior ∝ Likelihood × Prior    p(θ|X) ∝ p(X|θ) · p(θ)
Conjugate Priors
LikelihoodPriorPosterior
BinomialBeta(α,β)Beta(α+x, β+n-x)
PoissonGamma(α,β)Gamma(α+Σx, β+n)
Normal (known σ²)Normal(μ₀,σ₀²)Normal (updated)
Bayesian vs Frequentist
Frequentist

Parameters are fixed, unknown constants. Probability = long-run frequency. Uses MLE, CIs, p-values.

Bayesian

Parameters have probability distributions. Probability = degree of belief. Uses posterior, credible intervals.

Markov Chain Monte Carlo (MCMC)

Sample from complex posteriors when no closed form exists. Algorithms: Metropolis-Hastings, Gibbs sampling, Hamiltonian MC (used in PyMC, Stan).

Bayesian Neural Networks

Place distributions over weights: p(w|D) ∝ p(D|w)p(w). Provides uncertainty estimates — crucial for safety-critical AI systems.

8Information Theory
Entropy
H(X) = -Σ p(x) log₂ p(x)    [bits]   or   -Σ p(x) ln p(x)  [nats]

Measures uncertainty / information content. Uniform distribution → maximum entropy. Used in decision tree splitting (ID3, C4.5).

Cross-Entropy
H(p, q) = -Σ p(x) log q(x)

Loss function for classification in deep learning. Minimizing cross-entropy ≡ maximizing likelihood.

KL Divergence
D_KL(p ‖ q) = Σ p(x) log [p(x)/q(x)] ≥ 0

Not symmetric. Measures how much q differs from p. Used in VAEs, reinforcement learning (PPO), NLP (RLHF).

H(p, q) = H(p) + D_KL(p ‖ q)
Mutual Information
I(X;Y) = H(X) - H(X|Y) = H(Y) - H(Y|X)

Measures statistical dependence between two variables. Used for feature selection. I(X;Y) = 0 iff X,Y independent.

Gini Impurity (Classification Trees)
Gini = 1 - Σᵢ pᵢ²    (used in CART algorithm)
9Dimensionality Reduction
Principal Component Analysis (PCA)

Finds orthogonal directions of maximum variance in data. Reduces dimensions while preserving most information.

Steps: Standardize → Covariance matrix Σ = (1/n)XᵀX → Eigen-decomp → Sort eigenvalues → Project
Variance explained by component k: λₖ / Σλᵢ
Singular Value Decomposition (SVD)
X = UΣVᵀ    (U: left singular vectors, Σ: singular values, V: right singular vectors)

Foundation of PCA, recommender systems (matrix factorization), NLP (LSA). Truncated SVD for dimensionality reduction.

Linear Discriminant Analysis (LDA)

Supervised dimensionality reduction. Maximizes between-class to within-class variance ratio. Used in classification preprocessing.

J(W) = |Wᵀ S_B W| / |Wᵀ S_W W|
Manifold Learning Methods
t-SNE (visualization) UMAP (topology) Isomap (geodesic) Autoencoder (nonlinear)
Curse of Dimensionality

As dimensions increase: data becomes sparse, distances become similar, volume grows exponentially. Regularization and dimensionality reduction are essential antidotes in high-dimensional ML.

Cross-Validation
k-Fold CV

Split data into k folds. Train on k-1, test on 1. Rotate. Average scores. Standard: k=5 or k=10.

Stratified k-Fold

Maintains class proportions in each fold. Essential for imbalanced datasets.

Leave-One-Out (LOO)

k = n. High variance, computationally expensive. Best for tiny datasets.

Time-Series CV

Forward chaining. Training set always precedes test. Prevents data leakage.

Model Selection Criteria
AIC = 2k - 2ln(L̂)    (penalizes parameters)
BIC = k·ln(n) - 2ln(L̂)    (stronger penalty for large n)

Lower AIC/BIC = better model. BIC prefers simpler models than AIC as n grows.

Bootstrap & Resampling
Draw n samples with replacement from dataset of size n. Repeat B times. Estimate statistic distribution.

Used for confidence intervals of model performance, ensemble methods (bagging), random forests.

Key principle: Always evaluate on a held-out test set never seen during training or hyperparameter tuning. Use a three-way split: train / validation / test.
